CLONE X LOCAL WEBSITE
=====================

Open the website:
  Double-click START-CLONEX.cmd

Close the local server:
  Double-click STOP-CLONEX.cmd

Local address:
  http://127.0.0.1:4173/

Keep this entire folder together. All 3D models, textures, fonts, videos,
audio and interface assets are stored locally and do not require the
original CloneX website.

Do not open index.html directly with a file:// address. Web browsers block
local WebGL models and dynamic JavaScript modules in that mode.
